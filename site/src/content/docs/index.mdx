docs cover the citation pipeline, the agentic surfaces, and the
manuscript-as-software CI.
</Aside>

quartobot wires citation grounding into Quarto. Authors write
`@doi:10.21105/joss.01686` or `@pmid:23685459` (or `@arxiv:`, `@isbn:`,
`@url:`, `@wikidata:`, `@pmc:`) directly in their prose; a pre-render
hook resolves each key to canonical metadata before pandoc runs, the
manuscript builds itself on every commit, and every commit gets an
immutable permalink at `/v/<sha>/` on GitHub Pages.

The same resolver works the other direction too: an agent in Claude
Desktop, Codex, or Gemini Code Assist (over MCP) or a shell-tool
agent (over `quartobot resolve --output -`) can ground citations as
it drafts, against the same call the renderer uses. The agent's
references and the rendered manuscript's references can't drift.

    `validate_project` as MCP tools so an agent in Claude Desktop,
    Codex, or Gemini Code Assist can call the same resolver your

    [MCP server →](./mcp/)

    Anthropic-SDK harnesses, in-house automation) pipe through `jq`

## Modular by design

Three reach-points into one resolver. The pre-render hook is for
render-time citation resolution; the MCP server is for agent-native
clients; the stdout CLI is for everything else with a shell. Same
underlying call, no parallel implementations, no fork. You pick the
surface that fits where you already live.

## Origin

[manubot/manubot#332](https://github.com/manubot/manubot/issues/332)
("Quarto integration") was opened by Anthony Gitter in April 2022 after
a conversation with Sean Davis. quartobot resolves that issue. The
project depends on [manubot](https://github.com/manubot/manubot) for
the citation resolver (`manubot.cite.citekey_to_csl_item` — eight years
of accumulated source-API quirks) and has diverged since on
architecture, ergonomics, and the agent-facing surfaces.

## Working example

[seandavi/2026-venice-spatial-hackathon-manuscript](https://github.com/seandavi/2026-venice-spatial-hackathon-manuscript)
runs the CI / permalink / banner half on a live 25-author preprint
from the Bioconductor Spatial Hackathon. The CI machinery `quartobot
use github-ci` scaffolds is being lifted from there.
